Mathematical Identities and Chemical Equations

Description:

This webpage displays a collection of common algebraic formulas and basic chemical reactions using HTML superscript and subscript formatting. It demonstrates the correct representation of mathematical identities and chemical equations, along with navigation buttons for easy access to other pages.

Mathematical Identities

  1. (x + y)2 = x2 + y2 + 2xy
  2. (a − b)2 = a2 − 2ab + b2
  3. a2 + b2 = (a + b)2 − 2ab
  4. (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca
  5. (a + b)3 = a3 + b3 + 3ab2 + 3a2b

Chemical Equations

  1. 2H2 + O2 → 2H2O
  2. S8 + 12O2 → 8SO3
  3. 2Na + 2H2O → 2NaOH + H2
